Use position classes to control the offcanvas panel's placement.
- offcanvas-top for a top-positioned panel.
- offcanvas-end for a right-side panel.
- offcanvas-bottom for a bottom-positioned panel.
- offcanvas-start for a left-side panel.
The showOffcanvas() method dynamically sets the position and visibility.
