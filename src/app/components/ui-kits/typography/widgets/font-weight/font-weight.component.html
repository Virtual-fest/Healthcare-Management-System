Using f-w-* class, you can change the font weight.

You can set bolder font weight Heading 1 using font-black

You can set bold font weight Heading 2 using font-bold

You can set medium font weight Heading 3 using font-semibold

You can set normal font weight Heading 4 using font-medium

You can set light font weight Heading 5 using font-normal
You can set light font weight Heading 6 using font-light
